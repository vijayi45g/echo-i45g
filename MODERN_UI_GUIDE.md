## 🚀 How to Run

### **Step 1: In VS Code Terminal**

```bash
cd ~/Documents/i45g/pc-monitoring/network-monitoring/backend
```

### **Step 2: Build the server**

```bash
go build -o server main.go database.go
```

### **Step 3: Run the server**

```bash
./server
```

**You should see:**
```
INFO: Database connected successfully at monitoring.db
INFO: Database tables initialized  
Server running at http://localhost:8081
```

**IMPORTANT: Keep this terminal open!**

---

### **Step 4: Open your browser**

```
http://localhost:8081
```
